# battle-simulator
Command line battle simulator written in pure-Python.

## Mini-documentation:

Instantiate an enemy:

    Syntax: 
    '''python
        enemy = GenerateMonster(str name, int health, int gold, str weapon, int ac)
    '''
    Example:
    '''python
        goblin = GenerateMonster('Goblin King', 1000, 100, 'fists', 15)
    '''
